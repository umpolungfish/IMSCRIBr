cfg_sequences.py: animated token-sequence CFGs for all 12 canonical IMASM classes

New script: cfg_sequences.py

For each of the 12 canonical IMASM arrangement classes (I–XII), generates
an animated GIF in docs/cfg_seq_{class_key}.gif:

  Visualization:
    - nodes coloured by token family:
        LOGICAL=#4e79a7  FROBENIUS=#ffd700  DIALETHEIA=#e15759  LINEAR=#59a14f
    - register state annotated below each node (VOID/T/F/B⬡)
    - ouroboric back-arc when ouroboricity > O₀
    - Everson Mono font for Shavian IG-type footer
    - Phase 1: build — nodes appear step by step
    - Phase 2: flow — Gaussian pulse travels the sequence

  Register simulation:
    - full dialetheic-aware IMASM machine (inline, no imports)
    - pre-split EVALT tracking for FFUSE dialetheic mode
    - IFIX/IMSCRIB exception correctly handles fixed-register steps

  Sequences: all 12 from CANONICAL_CLASSES:
    I Dialetheic Bootstrap (O₂)  → B⬡ final
    II Void Genesis (O₀)          → T final
    III Anchor Protocol (O₁)      → T final
    IV Dual Bootstrap (O_∞)       → B⬡ final
    V Linear Chain (O₀)           → VO⌀ final
    VI Empty Bootstrap (O₁)       → T final
    VII Parakernel (O₂)            → T final
    VIII Frobenius Kernel (O₀)    → VO⌀ final  [4 steps]
    IX Chiral Pairs (O₁)          → B⬡ final
    X Truth Machine (O₁)          → T final
    XI Eternal Return (O₂)        → B⬡ final
    XII ROM Burn (O₀)             → T final
