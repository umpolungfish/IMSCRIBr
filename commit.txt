symbolic_diagram.py (pen mode): draw the ouroboric closure arc to the return anchor

Pen diagrams now always show the closure connection from the final token back to
the token the cycle re-enters at. The return anchor is not assumed to be the start
node: it is the first IMSCRIB (self-imscription / identity) if one exists and is not
the final node, otherwise the first node. So a VINIT-start canon correctly arcs back
to IMSCRIB rather than to VINIT.

  _pen_return_target(tokens, last, pos): resolves the anchor (first IMSCRIB else start).
  _pen_backarc(svg, src, tgt, label):    solid arc from the final node, arcing above
                                         the diagram, arrowhead pointing down into the
                                         anchor node; labeled with the ouroboricity.

Verified: for VINIT AFWD IMSCRIB FSPLIT EVALT EVALF FFUSE TANCH IFIX the arc runs
IFIX -> IMSCRIB (node 2), not IFIX -> VINIT; for the IMSCRIB-bookended canon it runs
last IMSCRIB -> first IMSCRIB (node 0).
