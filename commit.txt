wiring.py: port-graph cross-branch model + novel sequences XX–XXIII
engine.py: search_cross_branch_wirings(); cfg_dag.py: wired DAG renderer

wiring.py (new):
  Full port-level graph model for IMASM token arrangements. Every token is a
  node with typed ports: VINIT=(out:o), FSPLIT=(in:i, out:T/F), FFUSE=(in:T/F,
  out:o), all others=(in:i, out:o).

  Wire, WiredGraph: frozen-slot wire class; WiredGraph dataclass with validate(),
  topological_layers(), cross_branch_wires(), wire_y_hint().

  cross_branch_wires(): flags only FSPLIT→FFUSE arcs to non-default FFUSE;
  FSPLIT→FSPLIT nesting (outer T-wire routing into inner FSPLIT.i) excluded.

  imscr_wiring(): nesting-aware branch assignment. Depth computed per-token;
  each pair claims only tokens at its own depth. Fixes prior bug where outer
  pair stole inner-pair tokens (VINIT FSPLIT FSPLIT EVALT FFUSE EVALF FFUSE TANCH
  previously produced duplicate in-port error; now valid). FFUSE outputs use
  branch_internal-skip to find next reachable node regardless of nesting depth.

  cross_wiring_variants(): F-output permutation iterator; valid only when
  F-branches are empty (no orphaned tokens, no duplicate in-ports).

  NOVEL_GRAPHS — 4 hand-crafted cross-branch sequences, all validate() == []:
    XX   Branch Entangle:  X-topology, FSPLIT1.F↔FSPLIT2.F crossed. cross=2.
    XXI  Paradox Bridge:   ENGAGR outer-F bypasses inner block into inner FFUSE. cross=1.
    XXII Fold Back:        pure Logical+Frobenius X-topology, no dialetheia. cross=2.
    XXIII Möbius Fork:     IMSCRIB-bounded self-ref; ENGAGR+outer-F dual-source. cross=2.

engine.py:
  search_cross_branch_wirings(): scans frob_count >= 2*min_pairs signatures;
  returns (arrangement, valid_variants) for all cross-wirable arrangements.

cfg_dag.py:
  Added: _wired_layout, _wire_color, render_wired_dag_frame,
  generate_wired_dag_gif, main_wired (renders NOVEL_GRAPHS to GIFs).
  Added simulate_wired(graph): dataflow simulation following WiredGraph wire
  topology. Topological layer order; Belnap 4-valued token functions; FFUSE
  joins T/F port values via _belnap_join (∨); FSPLIT distributes input to both
  T and F outputs; cross-branch wires route values to actual port destinations.
  generate_wired_dag_gif calls simulate_wired() — not the flat simulate_register().

cfg_dag_momonados.py: DAG CFGs for mOMonadOS canonical programs

New script: cfg_dag_momonados.py

Generates DAG CFGs for the 12 mOMonadOS canonical programs (src/tokens.rs).
Extends cfg_dag.py with multi-segment decomposition: sequences with multiple
FSPLIT/FFUSE pairs render as consecutive fork blocks.

  Key differences from IMSCRIBr canonicals:
    II  Void Genesis:        VINIT FSPLIT EVALT FFUSE EVALF CLINK IFIX ISCRIB
    III Anchor Protocol:     TANCH AFWD EVALT AREV EVALF CLINK IFIX TANCH (linear)
    VII Parakernel:          ENGAGR AFWD FSPLIT EVALT FFUSE EVALF IFIX ENGAGR
    VIII Frobenius Kernel:   FSPLIT FFUSE FSPLIT FFUSE — 2 consecutive fork blocks
    XI  Eternal Return:      TANCH AFWD AREV × period (linear)

  VIII renders as ⟨ ∅ | ∅ ⟩ . ⟨ ∅ | ∅ ⟩: two adjacent Frobenius identity
  lenses — (μ∘δ)∘(μ∘δ) = id∘id = id — visually showing μ∘δ=id composed twice.

  Ouroboricity computed from mOMonadOS kernel's compute_tier logic:
    I, VII: O_∞  (dial + self_ref + frob, period ≥ 3)
    II, IV, VIII, X, XII: O₁
    III, V, VI, IX, XI: O₀

  Ouroboric arc fires on two conditions (not just ouro != O₀):
    1. ouroboricity > O₀ (as before)
    2. visual_period(seq) in (2, n-1) — catches cut-short repeating prefixes
       e.g. XI (1,2,3,1,2,3,1,2) has vp=3 though 8%3≠0; arc labeled "p=3"
    Constant sequences (vp=1) excluded — a flat IFIX run isn't "returning."

Output: docs/cfg_dag_momonados_{key}.gif (12 files)
Run: python3 cfg_dag_momonados.py

tokens.py, engine.py: DAG arity and unflattened dag_str renderer

tokens.py:
  - TOKEN_ARITY: (in-degree, out-degree) for each of the 12 tokens as DAG nodes.
    VINIT=(0,1) source; FSPLIT=(1,2) fork; FFUSE=(2,1) join; all others=(1,1).
  - TOKEN_BRANCH: {EVALT: 'T', EVALF: 'F'} — branch-constrained tokens.
  - FORK_TOKENS, JOIN_TOKENS: frozensets for structural detection.
  - dag_str(arr): renders an arrangement as an unflattened DAG expression.
      Sequential tokens: A . B . C
      Matched FSPLIT/FFUSE pair: ⟨ T-branch | F-branch ⟩
      Branch assignment: EVALT anchors T, EVALF anchors F; tokens before
      the first anchor follow whichever branch is present.
      Unmatched FSPLIT or inverted FFUSE→FSPLIT: rendered inline (flat).
      Recursive: post-fuse segment recurses to catch further fork blocks.

engine.py:
  - Imports TOKEN_ARITY, TOKEN_BRANCH, FORK_TOKENS, JOIN_TOKENS, dag_str.

All 12 canonical classes verified. Examples:
  I   Dialetheic Bootstrap: IMSCRIB . EVALT . ⟨ ∅ | EVALF ⟩ . ENGAGR . IFIX . IMSCRIB
  II  Void Genesis:         VINIT . TANCH . AFWD . ⟨ CLINK | ∅ ⟩ . IFIX . IMSCRIB
  VII Parakernel:           EVALF . AREV . ⟨ EVALT . AFWD | ∅ ⟩ . ENGAGR . IFIX
  VIII Frobenius Kernel:    VINIT . ⟨ ∅ | ∅ ⟩ . TANCH
